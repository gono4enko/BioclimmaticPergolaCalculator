## Общая архитектура

Калькулятор реализован в виде веб-приложения на Python с использованием фреймворка Streamlit. Основные компоненты:

- `app_basic.py` - основной файл приложения с логикой расчета и пользовательским интерфейсом
- `attached_assets/` - директория с прайс-листами в формате CSV
- `backups/` - директория с резервными копиями стабильных версий
- `docs/` - директория с документацией

## Основные функции

### Функции загрузки и обработки данных

- `load_price_data(pergola_type, lamella_size)` - загружает данные о ценах из соответствующего CSV файла
- `get_base_price(pergola_type, lamella_size, width_m, length_m)` - определяет базовую цену перголы

### Функции определения дополнительных опций

- `get_modules_by_dimensions(width, length, pergola_type=None)` - определяет количество модулей
- `needs_additional_columns(pergola_type, lamella_size, length_m)` - проверяет необходимость дополнительных колонн
- `needs_gutter_insert(length_m)` - проверяет необходимость усилителя лотка
- `calculate_gutter_insert_price(length_m, modules)` - рассчитывает стоимость усилителя лотка
- `get_drive_price(pergola_type, width_m, length_m, modules)` - определяет тип и стоимость привода
- `get_remote_control(devices_count)` - определяет тип и стоимость пульта управления
- `calculate_lighting_perimeter(width_m, length_m, modules=1)` - расчет периметра подсветки
- `adjust_length_for_lamella_size(length_m, lamella_size_mm)` - корректирует размер выноса перголы до целого числа ламелей

### Функции интерфейса

- `render_dimensions_form()` - отображает форму для ввода размеров перголы
- `render_options_form()` - отображает форму для выбора опций перголы
- `render_results(results)` - отображает результаты расчета
- `main()` - основная функция приложения, объединяющая все компоненты

### Основная функция расчета

- `perform_calculation(dimensions, options)` - главная функция, выполняющая все расчеты и формирующая результаты

## Структура данных

### Константы

В начале файла `app_basic.py` определены основные константы:

- `PERGOLA_TYPES` - словарь типов пергол
- `LAMELLA_TYPES` - словарь типов ламелей
- `DRIVE_PRICES` - словарь с ценами на приводы
- `REMOTE_CONTROL_PRICES` - словарь с ценами на пульты управления
- `LIGHTING_PRICES` - словарь с ценами на компоненты освещения
- `ADDITIONAL_COLUMNS_PRICE` - цена дополнительных колонн
- `GUTTER_INSERT_PRICE` - цена усилителя лотка за метр

### Входные данные

- **Размеры перголы**: ширина и вынос (длина) в метрах
- **Тип перголы**: B500NEW, B700NEW или B600
- **Тип ламелей**: зависит от типа перголы (200 мм, 250 мм или PIR)
- **Опции освещения**: белая и/или RGB светодиодная лента

### Выходные данные

Результаты расчета возвращаются в виде словаря со следующей структурой:

```python
{
    "dimensions": {"width": width, "length": length, "modules": modules},
    "options": {"pergola_type": pergola_type, "lamella_type": lamella_type, "lighting": lighting_options},
    "base_price": base_price,
    "total_price": total_price,
    "items": [{"name": "...", "price": ...}, ...],
    "specification": [{"name": "...", "count": "...", "price": ""}, ...],
    "debug": {...}  # Отладочная информация
}
```

## Интерфейс пользователя

Интерфейс пользователя разделен на несколько блоков:

1. **Заголовок** - название и краткое описание калькулятора
2. **Форма размеров** - поля для ввода ширины и выноса (длины) перголы
3. **Форма опций** - выбор типа перголы, типа ламелей и опций освещения
4. **Кнопка расчета** - запускает расчет и отображение результатов
5. **Результаты** - показывает общую информацию, спецификацию и стоимость

## Алгоритм расчета

Общий алгоритм расчета стоимости перголы:

1. **Получение входных данных**:
   - Размеры перголы (ширина и вынос/длина)
   - Тип перголы и тип ламелей
   - Дополнительные опции (освещение)

2. **Расчет базовой стоимости**:
   - Загрузка прайс-листа для выбранного типа перголы и размера ламелей
   - Определение базовой цены по размерам

3. **Определение количества модулей**:
   - На основе ширины и выноса (длины) перголы

4. **Определение необходимости дополнительных компонентов**:
   - Проверка необходимости дополнительных колонн
   - Проверка необходимости усилителя лотка

5. **Расчет стоимости автоматики**:
   - Определение типа и стоимости привода
   - Определение типа и стоимости пульта управления

6. **Расчет стоимости освещения** (если выбрано):
   - Расчет периметра для светодиодной ленты
   - Расчет стоимости компонентов освещения
   - Добавление блоков управления освещением (по 1 на каждый модуль)

7. **Формирование спецификации и итоговой стоимости**:
   - Создание списка компонентов с ценами
   - Суммирование всех компонентов для получения итоговой стоимости
