## Общая структура расчета

Калькулятор стоимости пергол использует следующую последовательность шагов для расчета:

1. Загрузка базовых цен из соответствующего прайс-листа в зависимости от типа перголы и размера ламелей
2. Автоматическое определение количества модулей на основе размеров перголы
3. Расчет необходимости дополнительных компонентов (колонны, усилители лотка)
4. Определение типа и стоимости привода и пульта управления
5. Расчет стоимости дополнительных опций (освещение)
6. Формирование итоговой спецификации и расчет общей стоимости

## Определение базовой стоимости перголы

Базовые цены на перголы содержатся в CSV-файлах:
- `Price_B500-20.csv` - для B500NEW с ламелями 200 мм
- `Price_B500-25.csv` - для B500NEW с ламелями 250 мм
- `Price_B700-20.csv` - для B700NEW с ламелями 200 мм
- `Price_B700-25.csv` - для B700NEW с ламелями 250 мм
- `Price_B600_PIR.csv` - для B600 с сэндвич-панелями PIR

Каждый файл содержит матрицу цен, где:
- Строки соответствуют различным значениям выноса перголы (длина в метрах)
- Столбцы соответствуют различным значениям ширины перголы (в метрах)
- На пересечении находится базовая стоимость перголы в евро

Калькулятор находит точное соответствие размерам в прайс-листе. Если точное соответствие не найдено, выводится сообщение об ошибке.

## Расчет количества модулей

Количество модулей определяется автоматически на основе размеров перголы по следующим правилам:

### Стандартные правила (по ширине):
- Для ширины до 4.5м - 1 модуль
- Для ширины 5-9м - 2 модуля
- Для ширины 9.5м и более - 3 модуля

### Правила для больших выносов (длины):
Если вынос перголы > 6.0м:
- Даже при малой ширине (до 4.5м) требуется минимум 2 модуля
- Для ширины 5-9м - 2 модуля
- Для ширины 9.5м и более - 3 модуля

## Расчет количества ламелей

Количество ламелей рассчитывается на основе длины (выноса) перголы и размера ламелей:

```
lamellas_count = round(length_m * 1000 / lamella_size_mm)
```

где:
- `length_m` - вынос перголы в метрах
- `lamella_size_mm` - размер ламели в миллиметрах (200 или 250)

Для B600 с панелями PIR количество ламелей не рассчитывается.

## Определение необходимости дополнительных колонн

Дополнительные колонны требуются при следующих условиях:

### Для перголы B500NEW:
- При ширине > 4м
- При выносе > 5м

### Для перголы B700NEW:
- При ширине > 4м
- При выносе > 6м

### Для перголы B600:
- При ширине > 4м
- При выносе > 5м

Стоимость дополнительных колонн рассчитывается по формуле:
```
columns_cost = ADDITIONAL_COLUMNS_PRICE * (modules + 1)
```

где `modules` - количество модулей перголы.

## Определение типа и стоимости привода

### Для перголы B500NEW:
- Для перголы шириной до 4м и выносом до 7м - привод Bansbach
- Для больших размеров (ширина > 4м или вынос > 7м) - привод Bansbach Tandem

### Для перголы B700NEW:
- Привод Somfy, цена зависит от размеров

### Для перголы B600:
- Привод не требуется (стационарные панели)

Для каждого типа привода указана фиксированная цена в константах. При наличии нескольких модулей стоимость привода умножается на количество модулей.

## Расчет стоимости освещения

Если выбрана опция освещения (белая или RGB светодиодная лента), расчет производится следующим образом:

1. Расчет периметра подсветки:
   - Для 1 модуля: `perimeter = 2 * (width + length)`
   - Для нескольких модулей: периметр рассчитывается с учетом внутренних перегородок

2. Стоимость светодиодной ленты:
   - Белая лента: `LIGHTING_PRICES["white_led"] * perimeter`
   - RGB лента: `LIGHTING_PRICES["rgb_led"] * perimeter`

3. Блоки управления освещением:
   - По 1 блоку управления Somfy RTS Dimmer на каждый модуль
   - Стоимость: `LIGHTING_PRICES["controller"] * modules`

## Определение типа пульта управления

Тип пульта определяется количеством устройств, которыми нужно управлять:

- Для 1 устройства - пульт Situo 1 RTS Pure II
- Для 2-3 устройств - пульт Situo 5 RTS Pure II
- Для 4 и более устройств - пульт Telis 16 RTS Pure

В количество устройств включаются:
- Модули привода перголы (для B500NEW и B700NEW)
- Блоки управления освещением (по 1 на каждый модуль)

Для пергол B600 требуется отдельный пульт только для управления освещением.

## Расчет стоимости усилителя лотка

Усилитель лотка требуется при выносе перголы > 6.5м.

Расчет количества лотков:
- Для 1 модуля - 2 лотка (по бокам)
- Для 2 модулей - 3 лотка (2 по бокам + 1 центральный)
- Для 3 модулей - 4 лотка (2 по бокам + 2 центральных)

Общая длина лотков: `length_m * gutters_count`

Стоимость усилителя: `GUTTER_INSERT_PRICE * total_gutter_length`
