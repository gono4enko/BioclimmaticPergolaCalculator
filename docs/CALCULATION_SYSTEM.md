## Общая архитектура системы расчета

Система расчета построена на модульной архитектуре, где каждый компонент отвечает за определенный аспект вычислений:

- **utils/price_loader.py** - загрузка и обработка данных из CSV-прайсов
- **utils/calculator.py** - основная логика расчета стоимости
- **utils/validation.py** - проверка корректности ввода и результатов расчета
- **config/price_data.py** - константы и дополнительные таблицы цен
- **config/price_calculation_rules.py** - правила и формулы расчета
- **config/pergola_types.py** - информация о типах пергол и ламелей

Процесс расчета включает следующие этапы:
1. Валидация введенных пользователем параметров
2. Определение базовой стоимости перголы по прайс-листу
3. Корректировка размеров перголы под технические требования
4. Определение количества модулей исходя из ширины
5. Выбор подходящего типа автоматики/привода
6. Расчет стоимости дополнительных опций
7. Вычисление итоговой стоимости с учетом всех компонентов

## Структура данных прайс-листов

### Формат CSV-прайсов

Прайс-листы хранятся в CSV-файлах в директории `data/`. Каждый файл соответствует определенному типу перголы и ламелей:
- `Price_B500-20.csv` - B500NEW с ламелями 200 мм
- `Price_B500-25.csv` - B500NEW с ламелями 250 мм
- `Price_B700-20.csv` - B700NEW с ламелями 200 мм
- `Price_B700-25.csv` - B700NEW с ламелями 250 мм
- `Price_B600_PIR.csv` - B600 с PIR панелями

### Структура файлов прайс-листов

Формат CSV-файлов:
```
modules,width,length,price,price_with_automation
1,3.0,4.05,8729,9429
1,3.0,4.45,9350,10050
...
2,6.0,4.05,17458,18858
...
```

Где:
- `modules` - количество модулей перголы
- `width` - ширина перголы в метрах
- `length` - вынос (длина) перголы в метрах
- `price` - базовая стоимость без автоматики в евро
- `price_with_automation` - стоимость с автоматикой в евро

## Алгоритм выбора базовой стоимости

Процесс выбора базовой стоимости из прайс-листа:

1. Загрузка всех подходящих прайс-листов для выбранного типа перголы и ламелей
2. Фильтрация строк, удовлетворяющих требованиям по размерам
   - Если точное соответствие не найдено, выбирается ближайший больший размер
3. Выбор оптимальной конфигурации с учетом количества модулей
   - При наличии нескольких вариантов с одинаковыми размерами предпочтение отдается варианту с меньшим количеством модулей
   - Если вариант с большим количеством модулей оказывается дешевле, выбирается он

Пример кода для выбора базовой стоимости:
```python
def find_optimal_price(pergola_type, lamella_type, width_m, length_m):
    csv_data = load_csv_price(pergola_type, lamella_type)
    
    # Фильтруем данные по размерам (ближайшие большие)
    suitable_rows = [row for row in csv_data if row['width'] >= width_m and row['length'] >= length_m]
    
    if not suitable_rows:
        return None, "Не найдены подходящие размеры в прайс-листе"
    
    # Сортируем по модулям, затем по цене
    suitable_rows.sort(key=lambda x: (x['modules'], x['price']))
    
    # Берем первую (оптимальную) конфигурацию
    optimal_config = suitable_rows[0]
    
    return optimal_config, None
```

## Расчет размеров и корректировок

### Корректировка размеров перголы

Для пергол типа B500NEW и B700NEW производится корректировка выноса (длины) для обеспечения целого числа ламелей:

1. Для ламелей 200 мм:
   - Шаг ламелей: 200 мм
   - Количество ламелей = вынос в мм / 200 (округляется вверх)
   - Корректированный вынос = количество ламелей * 200 мм

2. Для ламелей 250 мм:
   - Шаг ламелей: 250 мм
   - Количество ламелей = вынос в мм / 250 (округляется вверх)
   - Корректированный вынос = количество ламелей * 250 мм

### Учет технических ограничений

- Максимальная ширина перголы: 13.5 м
- Максимальный вынос перголы: 8 м
- Для перголы B500NEW:
  - С ламелями 200 мм: вынос требует округления до кратного 200 мм
  - С ламелями 250 мм: вынос требует округления до кратного 250 мм
- Для перголы B700NEW:
  - С ламелями 200 мм: вынос требует округления до кратного 200 мм
  - С ламелями 250 мм: вынос требует округления до кратного 250 мм

## Правила определения количества модулей

Количество модулей определяется на основе ширины перголы:

- Ширина до 4.5 м: 1 модуль
- Ширина от 4.5 м до 9.0 м: 2 модуля
- Ширина от 9.0 м до 13.5 м: 3 модуля

Пример кода для определения количества модулей:
```python
def determine_modules_count(width_m):
    if width_m <= 4.5:
        return 1
    elif width_m <= 9.0:
        return 2
    else:  # ширина до 13.5м
        return 3
```

## Выбор типа автоматики и привода

### Для перголы B500NEW (поворотные ламели)

1. **Bansbach T1** (стандартный) применяется, когда:
   - Ширина перголы ≤ 4.5 м (1 модуль)
   - Вынос перголы ≤ 6.5 м

2. **Bansbach T2** (усиленный) применяется, когда:
   - Ширина перголы ≤ 4.5 м (1 модуль)
   - Вынос перголы > 6.5 м

3. **Bansbach T1 TANDEM** (двойной стандартный) применяется, когда:
   - Ширина перголы > 4.5 м (2-3 модуля)
   - Вынос перголы ≤ 6.5 м

4. **Bansbach T2 TANDEM** (двойной усиленный) применяется, когда:
   - Ширина перголы > 4.5 м (2-3 модуля)
   - Вынос перголы > 6.5 м

### Для перголы B700NEW (сдвижные ламели)

1. **Somfy M1** (стандартный) применяется, когда:
   - Не выполняются условия для Somfy M2 или M2 TANDEM

2. **Somfy M2** (усиленный) применяется, когда:
   - Ширина перголы ≤ 9 м
   - Вынос перголы > 7 м

3. **Somfy M2 TANDEM** (двойной усиленный) применяется, когда:
   - (Ширина перголы > 9 м и вынос > 7 м) или (ширина > 10.5 м и вынос > 6 м)

### Выбор пульта ДУ

- Для перголы с 1 модулем: Simu 1K (1-канальный)
- Для перголы с 2 модулями: Simu 5K (5-канальный)
- Для перголы с 3 модулями: Simu 15K (15-канальный)

### Стоимость автоматики по типам

- Bansbach T1: 700 € за модуль
- Bansbach T2: 800 € за модуль
- Somfy M1: 700 € за модуль
- Somfy M2: 800 € за модуль
- Simu 1K: 25 €
- Simu 5K: 50 €
- Simu 15K: 75 €

## Расчет стоимости системы освещения

### Расчет длины светодиодной ленты

Общая длина светодиодной ленты рассчитывается по периметру перголы с учетом количества модулей:

1. Базовый расчет периметра = 2 × (ширина + вынос)
2. Количество лотков зависит от количества модулей:
   - 1 модуль: 2 лотка
   - 2 модуля: 3 лотка
   - 3 модуля: 4 лотка
3. Дополнительная длина ленты для лотков = (количество лотков - 1) × ширина
4. Итоговая длина ленты = базовый периметр + дополнительная длина

Пример кода для расчета:
```python
def calculate_lighting_perimeter(width_m, length_m, modules=1):
    """
    Расчет общего периметра освещения для перголы
    """
    base_perimeter = 2 * (width_m + length_m)
    
    # Количество лотков зависит от количества модулей
    gutters_count = modules + 1
    
    # Добавляем дополнительную длину для лотков (кроме крайних)
    additional_length = (gutters_count - 2) * width_m if gutters_count > 2 else 0
    
    return base_perimeter + additional_length
```

### Расчет стоимости подсветки

1. **LED подсветка**:
   - Стоимость ленты: 20 € за метр
   - Блок управления Somfy RTS Dimmer: 300 € за штуку
   - 1 блок управления на каждые 15 метров ленты

2. **RGB подсветка**:
   - Стоимость ленты: 30 € за метр
   - Блок управления Somfy RTS RGB: 400 € за штуку
   - 1 блок управления на каждые 15 метров ленты

Формула расчета:
```
Стоимость подсветки = (Длина ленты × Стоимость за метр) + (Количество блоков управления × Стоимость блока)
```

## Расчет дополнительных опций

### Дополнительные колонны

Дополнительные колонны требуются для пергол с большим выносом:
- Для ламелей 250 мм: выносе > 6.5 м
- Для ламелей 200 мм: выносе > 6.85 м

Стоимость дополнительных колонн зависит от ширины перголы:
- 300 € за каждые 4 метра ширины или меньше

```
Количество комплектов колонн = Округлить вверх(Ширина / 4)
Стоимость дополнительных колонн = Количество комплектов × 300 €
```

### Усилитель лотка

Для пергол с выносом более 6.5 метров требуется вставка для усиления лотка:
- Стоимость: 80 € за погонный метр ширины перголы

```
Стоимость усилителя лотка = Ширина перголы × 80 €
```

## Методика итогового расчета

Итоговая стоимость перголы рассчитывается по формуле:

```
Итоговая стоимость = Базовая стоимость + Стоимость автоматики + Стоимость пульта ДУ + 
                    Стоимость подсветки + Стоимость дополнительных колонн + 
                    Стоимость усилителя лотка + Доставка
```

Где:
- **Базовая стоимость** - цена перголы выбранных размеров из прайс-листа
- **Стоимость автоматики** - цена выбранного типа автоматики с учетом количества модулей
- **Стоимость пульта ДУ** - цена пульта, соответствующего количеству модулей
- **Стоимость подсветки** - если выбрана, рассчитывается по формулам выше
- **Стоимость дополнительных колонн** - если требуются, исходя из выноса перголы
- **Стоимость усилителя лотка** - если требуется, исходя из выноса перголы
- **Доставка** - 10% от суммы всех элементов перголы без установки

### Пример расчета

Для перголы B500NEW с ламелями 200 мм, шириной 4.0 м, выносом 6.0 м, с LED подсветкой:

1. Базовая стоимость (из прайс-листа): 10800 €
2. Количество модулей: 1 (ширина ≤ 4.5 м)
3. Тип автоматики: Bansbach T1 (стандартный, вынос ≤ 6.5 м)
   - Стоимость автоматики: 700 €
4. Пульт ДУ: Simu 1K (1 модуль)
   - Стоимость пульта: 25 €
5. LED подсветка:
   - Периметр: 2 × (4 + 6) = 20 м
   - Стоимость ленты: 20 м × 20 € = 400 €
   - Блоки управления: 2 блока × 300 € = 600 €
   - Общая стоимость подсветки: 1000 €
6. Дополнительные колонны: не требуются (вынос ≤ 6.5 м)
7. Усилитель лотка: не требуется (вынос ≤ 6.5 м)
8. Сумма всех элементов: 10800 + 700 + 25 + 1000 = 12525 €
9. Доставка: 12525 × 0.1 = 1252.5 €
10. Итоговая стоимость: 12525 + 1252.5 = 13777.5 € (округляется до 13778 €)

4. **test_prices.py** - проверка цен на дополнительные опции

### Валидация данных из прайс-листов

Для проверки корректности загрузки и использования данных из CSV-прайсов используется следующая методология:

1. Загрузка прайс-листа из файла
2. Выбор конфигурации для заданных размеров
