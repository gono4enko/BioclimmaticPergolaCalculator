# Документация калькулятора стоимости пергол DecoLife

## Общее описание

Калькулятор стоимости пергол - это веб-приложение на базе Python и Streamlit, позволяющее рассчитывать стоимость пергол различных типов (B500NEW, B700NEW, B600) с учетом размеров, опций и дополнительных компонентов.

## Версии и изменения

- Реализация автоматического определения количества модулей
- Расчет стоимости дополнительных опций (освещение, привод, пульт)

## Типы поддерживаемых пергол

1. **B500NEW** - пергола с поворотными ламелями
   - Доступные размеры ламелей: 200 мм, 250 мм
   - Поддерживает автоматическое управление (привод Bansbach)
   - Поддерживает светодиодное освещение

2. **B700NEW** - пергола со сдвижными ламелями
   - Доступные размеры ламелей: 200 мм, 250 мм
   - Поддерживает автоматическое управление (привод Somfy)
   - Поддерживает светодиодное освещение

3. **B600** - пергола с фиксированными PIR-панелями
   - Стационарные панели из сэндвич-материала PIR
   - Без автоматического управления
   - Поддерживает светодиодное освещение

## Доступные опции

- **Типы ламелей**: выбор между ламелями 200 мм и 250 мм для B500NEW и B700NEW
- **Освещение**: белая и/или RGB светодиодная лента
- **Автоматика**: приводы и пульты управления (зависит от типа перголы)

## Правила расчета размеров

- **Ширина**: 2.0 - 15.0 метров (с шагом 0.5 м)
- **Вынос (длина)**: 2.0 - 8.0 метров (с шагом 0.5 м)
- **Модули**: 1-3 модуля в зависимости от размеров (определяется автоматически)

## Запуск и использование

Калькулятор реализован на базе Streamlit и запускается командой:

```
streamlit run app_basic.py --server.port 5000
```

Для использования необходимо:
1. Выбрать размеры перголы (ширина и вынос)
2. Выбрать тип перголы (B500NEW, B700NEW или B600)
3. Выбрать тип ламелей (зависит от типа перголы)
4. При необходимости добавить опцию освещения
5. Нажать кнопку "Рассчитать стоимость"

После расчета отображаются:
- Итоговая стоимость
- Спецификация перголы
- Детальная разбивка стоимости по компонентам
