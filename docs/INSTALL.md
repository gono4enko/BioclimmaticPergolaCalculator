# Руководство по установке и запуску калькулятора пергол

## Версия 3.1

В этом руководстве описаны шаги по установке, настройке и запуску калькулятора стоимости пергол на локальном компьютере или сервере.

## Требования к системе

Для корректной работы калькулятора необходимы следующие компоненты:

- Python 3.10 или выше
- pip (менеджер пакетов Python)
- Около 100 МБ свободного места на диске
- Доступ к интернету для установки зависимостей
- Современный веб-браузер для доступа к интерфейсу (Chrome, Firefox, Safari, Edge)

## Пошаговая установка

### 1. Установка Python

Если Python еще не установлен, скачайте и установите его с официального сайта: https://www.python.org/downloads/

После установки проверьте, что Python успешно установлен, выполнив в командной строке:

```bash
python --version
# или
python3 --version
```

Убедитесь, что версия Python 3.10 или выше.

### 2. Клонирование репозитория

Скачайте исходный код калькулятора. Если используется система контроля версий Git:

```bash
git clone https://github.com/your-repository/pergola-calculator.git
cd pergola-calculator
```

Или просто скачайте и распакуйте ZIP-архив с кодом в удобное место.

### 3. Установка зависимостей

Установите необходимые библиотеки Python, запустив следующую команду:

```bash
pip install -r requirements.txt
```

или установите библиотеки вручную:

```bash
pip install streamlit pandas
```

### 4. Настройка конфигурации (при необходимости)

Для настройки поведения приложения Streamlit, создайте или отредактируйте файл `.streamlit/config.toml`:

```toml
[server]
headless = true
address = "0.0.0.0"
port = 5000

[theme]
primaryColor = "#0066cc"
backgroundColor = "#ffffff"
secondaryBackgroundColor = "#f8f9fa"
textColor = "#333333"
```

### 5. Проверка структуры файлов

Убедитесь, что в директории проекта присутствуют все необходимые файлы и директории:

- `app_basic.py` - основной файл приложения
- `attached_assets/` - директория с файлами прайс-листов и изображениями
- `logs/` - директория для хранения логов (будет создана автоматически)
- `backups/` - директория для резервных копий (будет создана автоматически)
- `docs/` - директория с документацией

### 6. Проверка прайс-листов

Убедитесь, что в директории `attached_assets/` присутствуют следующие CSV-файлы:

- `Price_B500-20.csv` - прайс-лист для B500 с ламелями 200 мм
- `Price_B500-25.csv` - прайс-лист для B500 с ламелями 250 мм
- `Price_B700-20.csv` - прайс-лист для B700 с ламелями 200 мм
- `Price_B700-25.csv` - прайс-лист для B700 с ламелями 250 мм
- `Price_B600_PIR.csv` - прайс-лист для B600 с PIR-панелями

## Запуск калькулятора

### Запуск локально

Для запуска калькулятора на локальном компьютере выполните:

```bash
streamlit run app_basic.py
```

После запуска в терминале появится сообщение с URL-адресом, по которому доступен интерфейс (обычно http://localhost:8501).

### Запуск на сервере

Для запуска калькулятора на сервере с доступом через интернет:

```bash
streamlit run app_basic.py --server.address=0.0.0.0 --server.port=5000
```

Это запустит приложение на порту 5000 с доступом со всех сетевых интерфейсов.

## Обновление данных о ценах

Для обновления данных о ценах необходимо заменить соответствующие CSV-файлы в директории `attached_assets/`:

1. Подготовьте новые CSV-файлы с актуальными ценами в том же формате
2. Замените существующие файлы в директории `attached_assets/`
3. Перезапустите приложение для применения изменений

Формат CSV-файлов для прайс-листов:

```
Вылет\Ширина (м);2.5;3.0;3.5;...
2.5;5431;6045;6659;...
3.0;5945;6638;7330;...
...
```

## Резервное копирование

Перед обновлением рекомендуется создать резервную копию текущей версии:

```bash
mkdir -p backups
cp app_basic.py backups/app_basic_v$(cat VERSION.txt).py
# Обновляем версию
echo "3.1" > VERSION.txt
echo "Бэкап создан: $(date)" >> backups/CHANGELOG.txt
```

## Настройка курса валюты

Для изменения курса евро к рублю, используемого при расчетах, необходимо отредактировать значение переменной `euro_rate` в файле `app_basic.py`:

```python
# Курс евро к рублю
euro_rate = 110.0  # Измените на актуальное значение
```

## Устранение проблем

### Ошибка импорта модулей

Если при запуске возникает ошибка импорта модулей, убедитесь, что все зависимости установлены:

```bash
pip install streamlit pandas
```

### Ошибки доступа к файлам

Если возникают ошибки доступа к файлам прайс-листов, проверьте наличие всех необходимых файлов и права доступа к ним:

```bash
ls -la attached_assets/*.csv
```

### Логирование

Для отладки проблем проверьте логи приложения:

```bash
cat logs/calculation_log.txt
```
