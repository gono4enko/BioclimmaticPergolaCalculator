## Системные требования

- Python 3.8 или выше
- Подключение к интернету для установки зависимостей
- 100 МБ свободного места на диске
- Любой современный веб-браузер (Chrome, Firefox, Safari, Edge)

## Установка зависимостей

### Установка Python

Если Python еще не установлен, загрузите и установите его с [официального сайта](https://www.python.org/downloads/).

### Установка необходимых библиотек

Откройте командную строку (терминал) и выполните следующую команду:

```bash
pip install streamlit pandas trafilatura
```

## Загрузка проекта

### Вариант 1: Клонирование из репозитория

```bash
git clone https://github.com/komfortny-dom/pergola-calculator.git
cd pergola-calculator
```

### Вариант 2: Загрузка ZIP-архива

1. Скачайте ZIP-архив с исходным кодом проекта
2. Распакуйте архив в удобное место на компьютере
3. Откройте командную строку и перейдите в распакованную папку:
   ```bash
   cd путь/к/папке/pergola-calculator
   ```

## Запуск приложения

### Базовая версия (оптимизированная)

```bash
streamlit run app_basic.py --server.port 5000
```

### Расширенная версия (если доступна)

```bash
streamlit run app.py --server.port 5000
```

## Доступ к приложению

После запуска приложения откройте веб-браузер и перейдите по адресу:

```
http://localhost:5000
```

Если порт 5000 уже занят, Streamlit автоматически выберет другой порт. В этом случае посмотрите на вывод в командной строке, где будет указан правильный адрес.

## Решение проблем

### Проблема: Не устанавливаются зависимости

**Решение**: 
- Убедитесь, что у вас установлена последняя версия pip:
  ```bash
  pip install --upgrade pip
  ```
- Попробуйте установить зависимости по отдельности:
  ```bash
  pip install streamlit
  pip install pandas
  pip install trafilatura
  ```

### Проблема: Приложение не запускается

**Решение**:
- Проверьте, что все зависимости установлены
- Убедитесь, что вы находитесь в корневой папке проекта
- Попробуйте запустить с явным указанием Python интерпретатора:
  ```bash
  python -m streamlit run app_basic.py
  ```

### Проблема: Файлы прайс-листов не найдены

**Решение**:
- Проверьте, что в папке `attached_assets` есть файлы прайс-листов в формате CSV
- Если файлы отсутствуют, загрузите их из резервной копии или обратитесь к администратору
