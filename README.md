# Калькулятор стоимости пергол v4.3.5 от 11.04.2025

## Информация о последней версии
Версия 4.3.5 содержит улучшения для мобильных устройств с фокусом на компактность формы и возможность просмотра всех элементов формы без необходимости прокрутки.

### Основные изменения в v4.3.5:
- Оптимизирована компактность формы на мобильных устройствах
- Удалены верхние отступы контейнера (padding-top: 0rem)
- Убраны отступы между секциями 'Выберите тип ламелей', 'Освещение' и 'Установка'
- Уменьшены отступы в мобильной версии до 0px между элементами
- Оптимизирован чекбокс 'С установкой' для лучшей видимости

### Предыдущие изменения в v4.3.4:
- Увеличены размеры всех заголовков в интерфейсе для лучшей читаемости
- Обновлена стилизация заголовков (увеличен размер с 1.8rem до 2.0-2.2rem)
- Добавлен более заметный синий цвет заголовков (#0066cc)
- Исправлена проблема с отображением изображений (замена use_container_width)
- Увеличен вес шрифта заголовков (font-weight: 600) для лучшей видимости

## Инструкции по использованию
1. Выберите размеры перголы (ширина и вынос)
2. Выберите тип перголы и ламелей
3. Выберите опции освещения (при необходимости)
4. Укажите, нужна ли установка
5. Нажмите "Рассчитать стоимость"

## Особенности мобильной версии
- Все элементы формы помещаются на экране без необходимости прокрутки
- Уменьшены размеры шрифтов и отступы для лучшей компактности
- Оптимизированы таблицы результатов для корректного отображения на малых экранах
