# Калькулятор стоимости пергол "Комфортный дом" v3.0

![Комфортный дом Logo](generated-icon.png)

Профессиональный инструмент для расчета стоимости пергол различных типов с учетом размеров, материалов и дополнительных опций, с выводом результатов в рублях.

## Описание

Калькулятор стоимости пергол — это веб-приложение, позволяющее быстро и точно рассчитать стоимость пергол моделей B500NEW, B700NEW и B600 с учетом выбранных пользователем параметров:
- Размеры конструкции (ширина, вынос, высота)
- Тип перголы (B500NEW, B700NEW, B600)
- Тип ламелей (200 мм, 250 мм, PIR-панели)
- Наличие и тип освещения (без освещения, LED, RGB)
- Опция установки (с наценкой 10%)

Приложение автоматически подбирает оптимальный тип привода, рассчитывает количество модулей и другие технические параметры, необходимые для корректного формирования стоимости. Все цены отображаются в рублях по актуальному курсу евро (110 ₽).

## Основные возможности

- Расчет стоимости пергол трех моделей (B500NEW, B700NEW, B600) в рублях
- Автоматическая наценка за доставку (10%)
- Опциональная наценка за установку (10%)
- Поддержка различных типов ламелей (200 мм, 250 мм, PIR-панели)
- Интуитивно понятный пользовательский интерфейс с плиточным дизайном
- Автоматическая корректировка размеров перголы до технически возможных значений
- Автоматический подбор оптимального типа привода в зависимости от размеров
- Расчет стоимости дополнительных опций (подсветка, дополнительные колонны и др.)
- Формирование подробной спецификации перголы и детальной информации о стоимости
- Адаптивный дизайн для работы на различных устройствах, включая мобильные
- Бухгалтерский формат отображения цен с разделителями тысяч

## Технологии

- **Frontend & Backend**: Python (Streamlit)
- **Обработка данных**: Pandas
- **Хранение данных**: CSV-файлы с прайс-листами
- **Логирование**: Встроенная система логирования всех действий пользователя
- **Адаптивный дизайн**: CSS и HTML для корректного отображения на мобильных устройствах

## Структура проекта

- `app_basic.py` - основной файл приложения (оптимизированная версия)
- `app.py` - альтернативная версия с расширенным интерфейсом
- `components/` - компоненты пользовательского интерфейса
- `config/` - конфигурационные файлы и правила расчета
- `data/` - файлы прайс-листов в формате CSV
- `utils/` - утилиты для расчета стоимости и загрузки данных
- `docs/` - документация по проекту
- `backups/` - резервные копии стабильных версий
- `attached_assets/` - прайс-листы и другие ресурсы

## Начало работы

1. Установите зависимости:
```bash
pip install streamlit pandas trafilatura
```

2. Запустите основное приложение (базовая версия):
```bash
streamlit run app_basic.py --server.port 5000
```

3. Откройте браузер по адресу: `http://localhost:5000`
