# Калькулятор стоимости пергол

Интерактивное веб-приложение для расчета стоимости биоклиматических пергол различных типов с учетом размеров, материалов и дополнительных опций.

## Описание

Калькулятор пергол – это специализированное приложение для компаний, занимающихся проектированием и установкой архитектурных конструкций. Приложение позволяет быстро рассчитать стоимость перголы на основе различных параметров:

- **Тип перголы**: B500 (с поворотными ламелями), B700 (со сдвижными ламелями), B600 (с PIR-панелями)
- **Размеры**: Ширина и вынос (длина) конструкции
- **Материалы**: Алюминиевые ламели различных размеров
- **Дополнительные опции**: Типы освещения, система установки

Приложение автоматически определяет оптимальное количество модулей в зависимости от выбранных размеров, рассчитывает стоимость перголы и предоставляет подробное описание всех компонентов.

## Функциональность

- **Интерактивный интерфейс**: Удобный выбор параметров перголы
- **Автоматический расчет**: Определение количества модулей, типа привода и стоимости
- **Генерация коммерческого предложения**: Создание PDF-документа со спецификацией перголы
- **Информативное описание**: Подробная информация о типах пергол и их компонентах
- **Адаптивный дизайн**: Корректное отображение на устройствах различного размера

## Технологии

- **Python**: Основной язык программирования
- **Streamlit**: Фреймворк для создания интерактивного веб-интерфейса
- **Pandas**: Обработка данных о ценах из CSV-файлов
- **ReportLab/FPDF**: Генерация PDF-документов с коммерческими предложениями
- **HTML/CSS**: Форматирование описаний и улучшение пользовательского интерфейса

## Структура проекта

- `app_basic.py` - Основной файл приложения
- `config/pergola_descriptions.py` - Модуль с описаниями пергол
- `attached_assets/` - Директория с изображениями и файлами цен
- `generated_pdf/` - Директория для сохранения созданных PDF-документов
- `.streamlit/config.toml` - Файл конфигурации Streamlit

## Установка и запуск

### Предварительные требования

- Python 3.8 или выше
- Установленные библиотеки из файла `requirements.txt`

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск приложения

```bash
streamlit run app_basic.py --server.port 5000
```

### Встраивание на сайт

Для встраивания калькулятора на сайт используйте HTML-код из файла `embed_code_for_tilda.html`. Код предназначен для блока T123 в Tilda, но может быть адаптирован для других CMS.

## Обслуживание и модификация

При внесении изменений в приложение следуйте инструкциям в файле `CONTENT_POLICY.md`, который определяет правила по сохранению оригинального контента и оформлению изменений.

## Политика содержимого

**Важно**: Категорически запрещено изменять оригинальные тексты описаний, пути к изображениям и другой контент, определенный в модуле `config/pergola_descriptions.py`. Подробная информация об этом содержится в файле `CONTENT_POLICY.md`.
