# Калькулятор стоимости пергол DecoLife

## Версия 3.5

Калькулятор стоимости пергол - это интерактивное веб-приложение, разработанное для расчета стоимости пергол различных моделей (B500NEW, B700NEW, B600) с учетом размеров, типа ламелей и дополнительных опций.

![Внешний вид калькулятора](attached_assets/Снимок%20экрана%202025-04-09%20в%2013.57.24.png)

## Основные функции

- Расчет стоимости пергол моделей B500NEW, B700NEW и B600
- Автоматическое определение количества модулей в зависимости от размеров
- Подбор оптимального типа привода и контроллеров
- Расчет стоимости дополнительных опций (освещение, доставка, установка)
- Генерация спецификации перголы
- Предоставление подробной информации о выбранной модели перголы

## Технологии

- **Backend**: Python 3.10+
- **Frontend**: Streamlit (интерактивный веб-интерфейс)
- **Данные**: CSV-файлы с прайс-листами
- **Валюта**: Конвертация цен из евро в рубли

## Быстрый старт

### Установка и запуск

1. Установите Python 3.10 или выше
2. Установите зависимости: `pip install streamlit pandas`
3. Запустите приложение: `streamlit run app_basic.py`

Подробные инструкции по установке и настройке можно найти в [docs/INSTALL.md](docs/INSTALL.md).

## Процесс расчета

1. Пользователь вводит размеры перголы и выбирает ее тип
2. Пользователь выбирает дополнительные опции (освещение, установка)
3. Система рассчитывает базовую стоимость перголы из прайс-листа
4. Система определяет количество модулей и тип привода
5. Система рассчитывает стоимость дополнительных опций
6. Формируется итоговая стоимость и спецификация перголы

## Обновление данных о ценах

Для обновления данных о ценах необходимо заменить соответствующие CSV-файлы в директории `attached_assets/`.
