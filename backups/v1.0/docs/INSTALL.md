# Инструкция по установке и запуску калькулятора пергол DecoLife

## Системные требования

- Python 3.10 или выше
- Минимум 2 ГБ свободной оперативной памяти
- Минимум 500 МБ свободного дискового пространства
- Подключение к интернету для установки зависимостей

## Подготовка окружения

### Установка Python

1. Скачайте и установите Python с официального сайта: https://www.python.org/downloads/
2. Убедитесь, что версия Python 3.10 или выше:
```bash
python --version
```

### Клонирование репозитория

1. Клонируйте репозиторий проекта:
```bash
git clone https://github.com/decolife/pergola-calculator.git
cd pergola-calculator
```

Или распакуйте архив с проектом в желаемую директорию.

### Установка зависимостей

1. Рекомендуется использовать виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
   - Windows:
   ```bash
   venv\Scripts\activate
   ```
   - Linux/Mac:
   ```bash
   source venv/bin/activate
   ```

3. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

Основные зависимости проекта:
- streamlit==1.29.0
- pandas==2.1.3
- numpy==1.26.2

## Запуск приложения

1. Убедитесь, что виртуальное окружение активировано.

2. Запустите приложение Streamlit:
```bash
streamlit run app.py
```

3. Приложение будет доступно по адресу: http://localhost:8501

## Настройка сервера

### Конфигурирование Streamlit для деплоя

Для развертывания приложения на сервере используйте следующую конфигурацию в файле `.streamlit/config.toml`:

```toml
[server]
headless = true
address = "0.0.0.0"
port = 5000
```

### Запуск в режиме сервера

```bash
streamlit run app.py --server.port 5000 --server.address 0.0.0.0
```

## Обновление прайс-листов

Для обновления данных о ценах:

1. Подготовьте CSV-файлы с новыми ценами в формате:
```
modules,width,length,price,price_with_automation
1,3.0,4.05,8729,9429
...
```

2. Замените файлы в директории `data/`:
   - `Price_B500-20.csv` - B500NEW с ламелями 200 мм
   - `Price_B500-25.csv` - B500NEW с ламелями 250 мм
   - `Price_B700-20.csv` - B700NEW с ламелями 200 мм
   - `Price_B700-25.csv` - B700NEW с ламелями 250 мм
   - `Price_B600_PIR.csv` - B600 с PIR панелями

## Резервное копирование

Перед внесением значительных изменений рекомендуется создать резервную копию:

```bash
mkdir -p backups/v{версия}
cp -r app.py components/ config/ data/ utils/ .streamlit/ backups/v{версия}/
echo "Дата создания: $(date)" > backups/v{версия}/VERSION.txt
echo "Версия: {версия}" >> backups/v{версия}/VERSION.txt
```

## Устранение неполадок

### Проблемы с запуском

1. **Ошибка импорта модулей**
   - Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
   - Проверьте структуру проекта на соответствие указанной выше

2. **Ошибка при загрузке прайс-листов**
   - Проверьте наличие и формат CSV-файлов в директории `data/`
   - Убедитесь, что файлы имеют правильную структуру и кодировку UTF-8

3. **Ошибка порта при запуске**
   - Проверьте, не занят ли порт 8501 другим приложением
   - Используйте другой порт: `streamlit run app.py --server.port 8502`

### Проблемы с расчетами

1. **Неправильные результаты расчета**
   - Запустите тесты для проверки корректности расчетов:
     ```bash
     python test_calculator.py
     python test_price_validation.py
     ```
   - Проверьте прайс-листы на соответствие актуальным ценам

2. **Ошибка при выборе опций**
   - Проверьте файл `config/pergola_types.py` на корректность определения типов пергол и ламелей
   - Убедитесь, что в `config/price_data.py` указаны правильные цены на опции
