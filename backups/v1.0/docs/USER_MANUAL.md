## Введение

Калькулятор стоимости пергол DecoLife - это инструмент для расчета стоимости пергол различных типов с учетом выбранных размеров и опций. Данное руководство поможет вам использовать все возможности калькулятора для получения точных расчетов.

### Поддерживаемые типы пергол

Калькулятор поддерживает следующие типы пергол:
- **B500NEW** - Биоклиматическая пергола с поворотными ламелями
- **B700NEW** - Биоклиматическая пергола со сдвижными ламелями
- **B600** - Стационарная пергола с PIR-панелями

## Начало работы

Для начала работы с калькулятором перейдите по адресу веб-приложения. Интерфейс калькулятора разделен на несколько блоков, расположенных последовательно:

1. Блок ввода размеров
2. Блок выбора типа перголы
3. Блок выбора типа ламелей
4. Блок выбора освещения
5. Кнопка "Рассчитать стоимость"
6. Блок результатов расчета (появляется после нажатия на кнопку)

## Ввод размеров перголы

В первом блоке введите требуемые размеры перголы:

1. **Ширина** - ширина перголы в метрах (от 2.0 до 13.5 м)
2. **Вынос** - вынос (длина) перголы в метрах (от 2.0 до 8.0 м)
3. **Высота** - высота перголы в метрах (от 2.0 до 4.0 м)

![Блок ввода размеров](../attached_assets/block_dimensions.png)

**Важно:** 
- Для пергол B500NEW и B700NEW вынос будет автоматически скорректирован до ближайшего значения, кратного шагу ламелей
- Для ламелей 200 мм шаг составляет 200 мм
- Для ламелей 250 мм шаг составляет 250 мм

## Выбор типа перголы

Во втором блоке выберите тип перголы, нажав на соответствующую плитку:

- **B500NEW** - Биоклиматическая пергола с поворотными ламелями (ламели поворачиваются вокруг своей оси)
- **B700NEW** - Биоклиматическая пергола со сдвижными ламелями (ламели сдвигаются в сторону)
- **B600** - Стационарная пергола с PIR-панелями (фиксированная крыша из PIR-панелей)

![Блок выбора типа перголы](../attached_assets/block_pergola_type.png)

## Выбор типа ламелей

В третьем блоке выберите тип ламелей (доступные варианты зависят от выбранного типа перголы):

### Для B500NEW:
- **Ламели 200 мм** - поворотные ламели размером 200 × 56 мм
- **Ламели 250 мм** - поворотные ламели размером 250 × 53 мм

### Для B700NEW:
- **Ламели 200 мм** - сдвижные ламели размером 200 × 56 мм
- **Ламели 250 мм** - сдвижные ламели размером 250 × 53 мм

### Для B600:
- Для этого типа перголы используются только PIR-панели

![Блок выбора типа ламелей](../attached_assets/block_lamella_type.png)

## Выбор освещения

В четвертом блоке выберите тип освещения для перголы:

- **Без освещения** - пергола без встроенного освещения
- **LED подсветка** - стандартная светодиодная подсветка с белым светом
- **RGB подсветка** - светодиодная подсветка с изменяемым цветом

![Блок выбора освещения](../attached_assets/block_lighting.png)

## Расчет стоимости

После ввода всех необходимых параметров нажмите кнопку "Рассчитать стоимость". Система выполнит расчет и автоматически прокрутит страницу к результатам.

![Кнопка расчета](../attached_assets/button_calculate.png)

## Интерпретация результатов

После расчета вы увидите блок результатов, состоящий из двух таблиц:

### Спецификация перголы

В этой таблице отображается информация о выбранной конфигурации перголы:

- **Тип перголы** - выбранный тип перголы (B500NEW, B700NEW или B600)
- **Тип ламелей** - выбранный тип ламелей (200 мм, 250 мм или PIR-панели)
- **Ширина** - ширина перголы в метрах
- **Вынос** - вынос (длина) перголы в метрах
- **Количество ламелей** - количество ламелей в конструкции
- **Площадь** - общая площадь перголы в квадратных метрах
- **Количество модулей** - количество модулей перголы (1-3 в зависимости от ширины)
- **Компоненты автоматики** - информация о типе привода и пульте управления
- **Подсветка** - тип и детали выбранной подсветки (если выбрана)

![Таблица спецификации](../attached_assets/table_specification.png)

### Стоимость

В этой таблице отображается детальная информация о стоимости:

- **Базовая стоимость конструкции** - стоимость перголы без опций
- **Стоимость дополнительных колонн** - стоимость дополнительных опорных колонн (если требуются)
- **Стоимость усилителя лотка** - стоимость усиления лотка (если требуется)
- **Стоимость автоматики** - стоимость системы автоматического управления и пульта ДУ
- **Доставка (10%)** - стоимость доставки (10% от суммы всех элементов)
- **ИТОГО** - итоговая стоимость перголы в евро

![Таблица стоимости](../attached_assets/table_price.png)

## Технические особенности расчета

При расчете стоимости система автоматически:

1. Определяет необходимое количество модулей исходя из ширины перголы
2. Корректирует вынос перголы до целого числа ламелей
3. Подбирает оптимальный тип привода в зависимости от размеров перголы
4. Рассчитывает длину светодиодной ленты и количество блоков управления
5. Определяет необходимость дополнительных колонн и усилителя лотка
6. Вычисляет стоимость доставки (10% от суммы всех элементов)

## Часто задаваемые вопросы

### Почему меняется вынос перголы после расчета?

Для пергол B500NEW и B700NEW вынос корректируется до целого числа ламелей. Например, если шаг ламелей 200 мм, а заданный вынос 4.1 м, система скорректирует его до 4.2 м (21 ламель).

### Почему автоматически определяется тип привода?

Тип привода выбирается автоматически в зависимости от размеров перголы для обеспечения оптимальной работы конструкции. Для перголы B500NEW используются приводы Bansbach T1 или T2, для B700NEW - Somfy M1 или M2.

### Что такое "количество модулей"?

Количество модулей определяет конструктивные особенности перголы и зависит от ее ширины:
- Ширина до 4.5 м: 1 модуль
- Ширина от 4.5 до 9.0 м: 2 модуля
- Ширина от 9.0 до 13.5 м: 3 модуля

### Как рассчитывается стоимость подсветки?

Стоимость подсветки включает:
1. Стоимость светодиодной ленты (20 € за метр для LED или 30 € за метр для RGB)
2. Стоимость блоков управления (300 € за блок для LED или 400 € за блок для RGB)

Длина ленты рассчитывается по периметру перголы с учетом внутренних лотков.

### Когда требуются дополнительные колонны?

Дополнительные колонны требуются для пергол с большим выносом:
- Для ламелей 250 мм: при выносе > 6.5 м
- Для ламелей 200 мм: при выносе > 6.85 м

### Когда требуется усилитель лотка?

Усилитель лотка требуется для всех пергол с выносом более 6.5 м.
