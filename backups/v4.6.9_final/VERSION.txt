Version: 4.6.9

Дата обновления: 13.04.2025
Основные изменения:
- Улучшен пользовательский интерфейс и исправлены проблемы с изображениями:
  • Увеличена ширина кнопки "РАССЧИТАТЬ СТОИМОСТЬ" для отображения текста в одну строку
  • Добавлены CSS-стили для предотвращения переноса текста на кнопке
  • Исправлена проблема отображения изображения перголы B700 со сдвижными ламелями
  • Обновлены пути к изображениям моделей пергол с кириллических на латинские имена
  • Оптимизировано отображение интерфейса на мобильных устройствах

- Предыдущие изменения (4.6.8):
- Улучшен функционал управления прайс-листами в административной панели:
  • Исправлена ошибка "No columns to parse from file" при загрузке CSV-файлов с проверкой структуры и формата
  • Добавлена возможность массового удаления прайс-листов с помощью чекбоксов, с группировкой по типам пергол и размерам ламелей
  • В выпадающем меню выбора прайс-листов теперь отображается дата загрузки файла
  • Добавлена проверка формата CSV для повышения надежности загрузки файлов
  • Улучшена обработка ошибок при загрузке CSV-файлов с использованием параметра on_bad_lines='skip'
  • Реализован интуитивно понятный интерфейс для массового управления прайс-листами

- Оптимизация размера проекта для надежного деплоя:
  • Уменьшен общий размер проекта с 12GB до 1.8GB для ускорения деплоя
  • Оптимизирована директория backups (уменьшение с 9.7GB до 67MB) с сохранением только 4 последних бэкапов
  • Создан скрипт backup_manager.py для автоматического контроля размера бэкапов
  • Добавлен файл .replignore для исключения объемных директорий из процесса деплоя
  • Оптимизированы настройки workflow для запуска основного файла app.py

- Улучшена навигация и пользовательский интерфейс:
  • Добавлены фиксированные кнопки "Изменить размеры" и "К результатам" справа вверху
  • Реализовано плавное перемещение к результатам расчета без перезагрузки страницы
  • Исправлена проблема с перекрытием выпадающего меню верхними кнопками
  • Обновлен JavaScript метод для надежной работы кнопок на всех устройствах
  • Улучшена совместимость с различными браузерами для позиционирования элементов
  • Устранена проблема дублирования кнопок при обновлении страницы

  • Исправлена центровка текста кнопки через flexbox стили
