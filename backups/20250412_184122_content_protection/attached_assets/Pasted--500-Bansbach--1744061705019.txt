Для управления открытием крыши пергола В500 применяется линейный электропривод Bansbach (Германия). 
Комплект привода для управления открытием крыши состоит из 3 элементов и всегда поставляется в таком виде:
1. Двигатель easyE-lift-50. Bansbanch, Germany 24VDC 4500Nm - 400 евро
2. Блок управления для 1-го двигателя + блок питания - 200 евро
3. Приемник - 100 евро
Назовем это комплектом автоматики «Bansbanch Т1, Germany» стоимостью 700 евро

Для управления большими переголами применяется привод easyE-lift-50 Tandem.
Состоит из:
1. Двигатель easyE-lift-50. Bansbanch, Germany24VDC 4500Nm - 2шт - 800 евро
2. Блок управления для 2-х двигателей + блок питания - 350 евро
3. Приемник - 100 евро
Назовем это комплектом автоматики «Bansbanch Tandem, Germany» стоимостью 1250 евро

Для каждого модуля перголы устанавливается один привод. Если по размерам подходит то Bansbanch Т1, Germany - 700 евро.
Если правила предписывают, то Bansbanch Tandem, Germany - 1250 евро

Теперь о правилах выбора привода. По умолчанию всегда применяем привод Bansbanch Т1, German. Его мощности при определенных размерах выноса может быт не достаточно. В таких случаях применяем привод Bansbanch Tandem, Germany
