## Установка и настройка

### Требования к окружению
- Python 3.9 или выше
- Пакеты из файла `pyproject.toml`

### Шаги установки

1. Клонируйте репозиторий:
   ```
   git clone [URL_репозитория]
   cd [название_директории]
   ```

2. Установите зависимости с помощью Poetry:
   ```
   poetry install
   ```
   
   Или с помощью pip:
   ```
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```
   streamlit run app_basic.py --server.port 5000
   ```

## Структура проекта

### Основные файлы и директории

- `app_basic.py` - Основной файл приложения с интерфейсом Streamlit
- `config/` - Директория с конфигурационными файлами
  - `pergola_descriptions.py` - Описания типов пергол и их характеристик
- `attached_assets/` - Изображения и файлы с данными о ценах
- `generated_pdf/` - Директория для сохранения генерируемых PDF-файлов
- `.streamlit/config.toml` - Конфигурация Streamlit
- `backups/` - Резервные копии стабильных версий

### Ключевые модули

- `pdf_generator_fpdf.py` - Генерация PDF-документов с коммерческими предложениями
- `web_scraper.py` - Модуль для работы с внешними данными
- `tools.py` - Вспомогательные функции и утилиты

## Обновление и модификация

### Политика содержимого

**ВАЖНО!** Перед внесением любых изменений ознакомьтесь с файлом `CONTENT_POLICY.md`, который содержит критически важные правила по сохранению оригинального контента.

### Обновление цен

1. Обновите CSV-файлы в директории `attached_assets/`
2. Проверьте соответствие формата новых файлов существующим
3. Тестируйте расчеты с помощью скриптов в директории `tests/`

### Добавление новых типов пергол

1. Добавьте описание нового типа в `config/pergola_descriptions.py`
2. Добавьте соответствующие CSV-файлы с ценами в `attached_assets/`
3. Обновите логику расчета в `app_basic.py` при необходимости
4. Тщательно тестируйте новую функциональность

## Публикация и встраивание

### Публикация на Replit

1. Убедитесь, что приложение корректно работает локально
2. Настройте `.streamlit/config.toml` для публикации:
   ```
   [server]
   headless = true
   enableCORS = true
   enableXsrfProtection = true
   address = "0.0.0.0"
   port = 5000
   ```
3. Настройте `workflows_config.toml`:
   ```
   [[run]]
   name = "PergolaMasterServer"
   command = "streamlit run app_basic.py --server.port 5000 --server.address 0.0.0.0"
   wait_for_port = 5000
   ```
4. Используйте кнопку "Deploy" в интерфейсе Replit

### Встраивание на веб-сайт

1. Используйте код из файла `embed_code_for_tilda.html` для встраивания на сайт Tilda
2. Для других CMS адаптируйте код, сохраняя основную структуру:
   ```html
   <iframe 
       src="https://[URL_вашего_приложения]" 
       width="100%" 
       height="650px" 
       frameborder="0"
       scrolling="no"
       onload="adjustIframeHeight(this)"
   ></iframe>
   <script>
   function adjustIframeHeight(iframe) {
     // Эта функция вызывается после загрузки iframe и позволяет
     // адаптировать высоту при изменении содержимого
     window.addEventListener('message', function(e) {
       if (e.data && e.data.type === 'iframe-height') {
         iframe.style.height = e.data.height + 'px';
       }
     });
   }
   </script>
   ```

## Резервное копирование

### Автоматическое резервное копирование

1. Используйте утилиту в директории `backups/`:
   ```
   python create_backup.py
   ```

### Ручное резервное копирование

1. Создайте копию основного файла приложения:
   ```
   cp app.py app.py.backup
   ```
2. Обновите файл `CHANGELOG.txt` с описанием изменений
3. Обновите версию в `VERSION.txt`
4. Обновите номер версии в самом приложении в строке:
   ```
   st.markdown("<div style='text-align: center; font-size: 0.7rem; color: #999;'>© 2025 Комфортный дом | Калькулятор пергол v4.X.X</div>", unsafe_allow_html=True)
   ```

## Часто задаваемые вопросы

### Как оптимизировать приложение для мобильных устройств?

Текущая версия (v4.3.5) уже оптимизирована для мобильных устройств. Применены следующие оптимизации:

1. Уменьшена высота разделителей секций до 10px (параметр `height` в функции `add_section_separator` в файле `animations.py`)
2. Уменьшена высота разделителей между формами до 3px
3. Добавлен отрицательный отступ над заголовком (-10px)
4. Уменьшены отступы вокруг подзаголовка
5. Исправлен тип параметра `animation_delay` в функции `display_formatted_description` для совместимости
6. Оформление строки "Итого" в таблице с жирным начертанием только для слова "Итого"

Если нужны дополнительные оптимизации, следуйте этим рекомендациям:

1. Уменьшите верхние отступы контейнера в CSS:
   ```css
   .block-container {
       padding-top: 0rem !important;
   }
   ```
   
2. Минимизируйте отступы между элементами формы:
   ```css
   @media (max-width: 768px) {
       div.stNumberInput, div.stTextInput, div.stSelectbox, div.stRadio, 
       div.stCheckbox, div.stSlider, div.stButton, div.stMultiselect {
           margin-top: 0px !important;
           margin-bottom: 0px !important;
       }
   }
   ```

3. Уменьшите размер шрифтов на мобильных устройствах:
   ```css
   @media (max-width: 768px) {
       .stApp, .stApp p, .stApp div, .stMarkdown {
           font-size: 0.9rem !important;
       }
   }
   ```

### Как изменить фирменные цвета приложения?

Отредактируйте секцию `[theme]` в файле `.streamlit/config.toml`:

```toml
[theme]
primaryColor = "#004B9A"  # Основной цвет
backgroundColor = "#FFFFFF"  # Фон
secondaryBackgroundColor = "#F0F2F6"  # Вторичный фон
textColor = "#262730"  # Цвет текста
```

### Где изменить формат PDF-документов?

Логика формирования PDF-документов находится в файле `pdf_generator_fpdf.py`. Для изменения шаблона редактируйте функцию `generate_commercial_offer()`.

### Как добавить новый тип перголы?

1. Добавьте описание в `config/pergola_descriptions.py`
2. Создайте соответствующие CSV-файлы с ценами
3. Дополните функцию `perform_calculation()` в файле `app_basic.py`
4. Обновите интерфейс выбора типа перголы
