# Промпт для Replit Agent — Маркетинговые улучшения калькулятора пергол

---

## КОНТЕКСТ ПРОЕКТА

Это Flask-приложение — калькулятор биоклиматических пергол. Стек: Flask + Jinja2 + ReportLab (PDF) + vanilla JS/CSS. Приложение развёрнуто на Replit.

**Файлы, которые НЕЛЬЗЯ трогать:**
- `calculator.py` — логика расчёта цен
- `app.py` — роуты Flask
- `static/data/*.json` — прайс-файлы
- `kp_content.py` — логика контента КП

**Файлы для изменения:**
- `templates/calculator.html` (или аналогичный шаблон) — основной UI калькулятора
- `pdf_generator.py` — генератор PDF-КП
- `static/css/` — стили
- `static/js/` — клиентский JS

**JS** — добавить функцию обновления счётчика. Функция должна:
1. Читать текущие значения формы: выбранный тип перголы, ширину, вынос
2. Вычислять приблизительную сумму через существующий API-эндпоинт `/calculate` (если он есть) или из формы
3. Отображать результат в `#price-bar-value` в формате `от X ₽`
4. Показывать/скрывать бар в зависимости от того, заполнены ли обязательные поля (тип + ширина + вынос)

```javascript
function updatePriceBar() {
  const priceBar = document.getElementById('price-bar');
  const priceBarValue = document.getElementById('price-bar-value');
  if (!priceBar || !priceBarValue) return;

  // Получить значения полей
  const pergolaType = document.querySelector('[name="pergola_type"]:checked, .pergola-type-card.selected');
  const width = parseFloat(document.querySelector('[name="width"]')?.value || 0);
  const depth = parseFloat(document.querySelector('[name="depth"]')?.value || 0);

  // Показывать бар только если выбран тип и введены размеры
  const hasType = !!pergolaType;
  const hasDims = width > 0 && depth > 0;

  if (!hasType || !hasDims) {
    priceBar.classList.add('price-bar--hidden');
    return;
  }

  priceBar.classList.remove('price-bar--hidden');

  // Отправить запрос на предварительный расчёт
  const formData = new FormData(document.querySelector('form') || document.getElementById('calculator-form'));
  
  fetch('/calculate', {
    method: 'POST',
    body: formData
  })
  .then(r => r.json())
  .then(data => {
    if (data && (data.total || data.price || data.total_price)) {
      const price = data.total || data.price || data.total_price;
      priceBarValue.textContent = 'от ' + Math.round(price).toLocaleString('ru-RU') + ' ₽';
    }
  })
  .catch(() => {
    // При ошибке скрыть бар чтобы не вводить в заблуждение
    priceBar.classList.add('price-bar--hidden');
  });
}

// Навесить обработчики на все поля формы
document.addEventListener('DOMContentLoaded', function() {
  const fields = document.querySelectorAll('input, select, [data-pergola-type]');
  fields.forEach(f => {
    f.addEventListener('change', updatePriceBar);
    f.addEventListener('input', updatePriceBar);
  });
  // Также обновлять при клике на карточки типов пергол
  document.querySelectorAll('.pergola-type-card, [data-model]').forEach(card => {
    card.addEventListener('click', () => setTimeout(updatePriceBar, 100));
  });
});
```

**ВАЖНО:** Если у роута `/calculate` нет JSON-ответа (только генерирует HTML), добавить в `app.py` новый роут `/calculate_preview` который принимает POST с параметрами `pergola_type`, `width`, `depth` и возвращает JSON `{"total": число}`. Используй существующую логику из `calculator.py`, только верни базовую цену без опций.

### 5. ФИЛЬТРАЦИЯ ОПЦИЙ ЗАПОЛНЕНИЯ ПРОЁМОВ ПО МОДЕЛИ ПЕРГОЛЫ

В блоке «Заполнение проёмов» карточки типов заполнений должны скрываться в зависимости от выбранной модели перголы.

**Правила совместимости:**
- `B500` — скрыть: FP-PIR
- `B700` — скрыть: FP-PIR  
- `B200` — скрыть: FP-PIR, W500, W600, W700, S500, S100
- `B600` — показать всё (FP-PIR доступна только здесь)

**Реализация:** добавить в HTML data-атрибуты на карточки заполнений:
```html
<!-- FP-PIR -->
<div class="facade-card" data-fill-type="fp-pir" data-compatible-models="b600">
  ...
</div>

<!-- W500/W600/W700 -->
<div class="facade-card" data-fill-type="w500" data-compatible-models="b500,b700,b600">
  ...
</div>

<!-- FZ-44, FP-20, S500, S100, ZIP — совместимы со всеми -->
<div class="facade-card" data-fill-type="fz44" data-compatible-models="b500,b700,b600,b200">
  ...
</div>
```

JS-функция фильтрации (добавить к существующему обработчику выбора типа перголы):
```javascript
function filterFacadeOptions(selectedModel) {
  const modelKey = (selectedModel || '').toLowerCase().replace(/[^a-z0-9]/g, '');
  
  document.querySelectorAll('[data-compatible-models]').forEach(card => {
    const compatible = card.dataset.compatibleModels.split(',');
    const isCompatible = compatible.some(m => modelKey.includes(m.replace(/[^a-z0-9]/g, '')));
    
    card.style.display = isCompatible ? '' : 'none';
    
    // Если карточка была выбрана и теперь скрыта — снять выбор
    if (!isCompatible && card.classList.contains('selected')) {
      card.classList.remove('selected');
    }
  });
}
```

Вызывать `filterFacadeOptions(selectedModelCode)` при каждом выборе типа перголы.
