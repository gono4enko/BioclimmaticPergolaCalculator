  var inp = document.getElementById('leadPhone');

    inp.classList.add('err');
    inp.focus();

## Как подставлять текст расчёта `window._calcText`

Это **самое важное** — текст расчёта нужно формировать в JS при нажатии кнопки «Рассчитать» и записывать в `window._calcText`. Пример:

```javascript
// Вызывается при нажатии кнопки "Рассчитать"
function calculate() {
  var width  = parseFloat(document.getElementById('width').value);
  var height = parseFloat(document.getElementById('height').value);
  var price  = width * height * PRICE_PER_SQM;

  // Формируем текст расчёта для мессенджеров и Telegram
  window._calcText = [
    '📐 Ширина: ' + width + ' м',
    '📐 Высота: ' + height + ' м',
    '💰 Стоимость: от ' + price.toLocaleString('ru') + ' ₽',
  ].join('\n');

  // Показываем результат пользователю
  document.getElementById('result').textContent = price.toLocaleString('ru') + ' ₽';

  // Показываем кнопку открытия формы
  document.getElementById('openLeadBtn').style.display = 'block';
}
```

## Важные замечания

1. **FormSubmit** — сервис используется для отправки email без бэкенда. Первый запрос на новый email потребует подтверждения — нужно проверить почту `zakaz@infopergola.ru`.

2. **ipapi.co** — геолокация по IP, бесплатно до 1000 запросов/день. Если лимит исчерпан — город будет «Не определён», это нормально.

3. **Порядок отображения**: кнопка «Обсудить расчёт» показывается только после того как пользователь нажал «Рассчитать». До расчёта кнопку не показывать.

4. **Max** — всегда открывает ссылку (href на кнопке не меняется), но параллельно fire-and-forget отправляет расчёт в Telegram менеджеру.

5. **WhatsApp/Telegram** — перехватываем клик через `e.preventDefault()`, формируем URL с текстом расчёта и открываем в новой вкладке.
