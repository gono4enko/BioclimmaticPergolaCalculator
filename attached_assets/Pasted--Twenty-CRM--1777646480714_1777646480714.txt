Задача: добавить интеграцию с Twenty CRM в калькулятор биоклиматических пергол.

=== КОНТЕКСТ ===

Калькулятор находится по адресу https://bioclimmatic.pergolarussia.ru/calculator
Встроен как iframe на https://pergolamarket.ru/calculator_bioclimatic_pergola

В конце расчёта показывается блок "Обсудить расчёт с менеджером" с кнопками:
- Telegram
- Max (мессенджер)  
- Перезвонить (отправляет заявку с номером телефона на почту)

Нужно добавить отправку заявки в Twenty CRM при нажатии кнопки "Жду звонка" 
(кнопка которая отправляет телефон для перезвонка).

=== ЧТО НУЖНО УЗНАТЬ ===

Прежде чем реализовать — ответь на следующие вопросы:

1. Какой стек используется в калькуляторе?
   - Flask/Jinja2 + vanilla JS?
   - Или React/Vue/другой фреймворк?

2. Где находится обработчик кнопки "Жду звонка"?
   - В каком файле Python/JS обрабатывается отправка телефона?
   - Что происходит после нажатия — AJAX запрос или POST форма?

3. Какие данные доступны в момент нажатия кнопки "Жду звонка"?
   - Номер телефона (вводит пользователь)
   - Тип перголы (B500/B700/B600/B200)
   - Размеры (ширина, вынос, высота)
   - Опции (подсветка, монтаж, заполнение проёмов)
   - Итоговая стоимость расчёта
   - URL страницы откуда открыт калькулятор (referer)

4. Есть ли уже отправка данных на внешние сервисы?
   - Как реализована отправка на email?
   - Есть ли уже webhook вызовы?

5. Какие переменные окружения уже используются?
   - Есть ли .env файл?
   - Как настроена отправка email?

=== ЧТО НУЖНО РЕАЛИЗОВАТЬ ===

После ответов на вопросы — реализовать следующее:

При нажатии кнопки "Жду звонка":
1. Получить номер телефона из поля ввода
2. Получить данные расчёта (тип перголы, размеры, стоимость)
3. Отправить POST запрос на наш webhook:
   POST https://erp.pergolarussia.ru/api/webhooks/calculator-lead
   {
     "phone": "+79001234567",
     "productType": "Биоклиматическая пергола",
     "name": "",
     "comment": "Тип: B500, Ширина: 4м, Вынос: 3м, Стоимость: 450000 руб",
     "source": "calculator_bioclim",
     "calculatorData": {
       "pergolaType": "B500",
       "width": 4,
       "depth": 3,
       "height": 2.5,
       "totalPrice": 450000,
       "options": ["led_white", "installation"]
     }
   }
4. Продолжить стандартную обработку (отправка email менеджеру — не менять)
5. Показать пользователю подтверждение "Менеджер свяжется с вами"
