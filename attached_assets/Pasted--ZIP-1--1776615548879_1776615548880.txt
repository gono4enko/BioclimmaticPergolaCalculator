Расчёт ZIP-маркиз — пошагово
1. Ввод и автопереключение типоразмера
Пользователь задаёт:

Конфигурация: ZIP 100 или ZIP 130
Ширина (м), Высота (м) — округляются вверх до шага 0.5 м для попадания в ячейку прайса
Ткань: Veozip (Serge Ferrari) / Soltis / Copaco
Цвет каркаса: RAL 9016 / 7024 / 9T08 / 8028 / Custom
Управление: электро / ручное
Привод (если электро): Simu / Somfy / Decolife
Монтаж: с установкой / без
Важно — автопереход ZIP 100 → ZIP 130:
Если выбран ZIP 100, но ширина >4 м или высота >3.5 м — система автоматически переключает на ZIP 130 (у ZIP 100 жёсткий лимит конструкции).

2. Базовая цена каркаса
Берётся из матрицы ZIP100 или ZIP130 в awning_pricing.json по округлённым размерам:

base = ZIP_table[width_round][height_round]   (в EUR)

Например, ZIP 100 при 2.0 × 2.0 м → 510 €.
Конвертируется в рубли: base × euro_rate (сейчас курс 100 ₽).

3. Ткань — солнцезащита и влагостойкость
Площадь рассчитывается как ширина × высота (м²). Дальше зависит от ткани:

Veozip (Screen Veosol/Veozip)
Базовая ткань, доплата 0 €/м².
Просто строка «Ткань: Screen Veosol / Veozip» с ценой 0.
Soltis (Serge Ferrari) — ПВХ ветро/водозащищённая
+15 €/м² для коллекций W88 и W96.
Артикул определяется по префиксу (W96-... → soltisW96, W88-... → soltisW88).
В смете: «Ткань: Soltis · ПВХ ветро/водозащищённое Serge Ferrari».
Copaco — Lunar Blackout (полная блокировка света и влаги)
+15 €/м² для коллекции copacoLunar.
В смете: «Ткань: Copaco · Lunar Blackout 0%».
Формула доплаты: fab_rate × ширина × высота × euro_rate.

Все доплаты настраиваются в админке _zip_fabric_sqm_surcharge_eur. Если поле обнулено по ошибке, защитный код _ensure_zip_fabric_surcharge_defaults восстанавливает значение по умолчанию 15 €/м².

4. Цвет каркаса
Цвет	Доплата
RAL 9016 / 7024 / 9T08 / 8028 (стандарт)	0%
Специальный RAL (custom)	+10% к базе
5. Привод (электроуправление)
Подбор зависит от габаритов проёма:

small = (ширина ≤ 3.5 м) И (высота ≤ 4.0 м)

Цена привода (EUR, без пульта) — из _motor_zip_body_costs:

Бренд	Маленький проём (small)	Большой проём
Simu	175 €	175 €
Somfy	180 € (somfy_small)	230 € (somfy_large)
Decolife	130 €	130 €
Как видите, Somfy — единственный бренд, у которого две модели приводов: маленький (для проёмов до 3.5×4.0 м) и большой (для крупных). Simu и Decolife используют один универсальный привод.

В смете: «Трубчатый электропривод Simu (ZIP)» или «… Somfy (ZIP)».

Для ZIP мы пока не выводим конкретную модель привода (например, «SIMU T6 RTS»), как это сделано для локтевых маркиз. Если хотите — могу добавить детализацию моделей по брендам, как у G500/G700.

6. Подбор пульта
Логика «умного» подбора пульта зависит от числа маркиз в заявке и наличия LED-подсветки.

Шаг 1 — расчёт нужных каналов:

ch_need = количество маркиз в заявке
        + 1 канал, если есть LED-подсветка

Для одной ZIP без LED → нужен 1 канал.

Шаг 2 — выбор минимального по цене подходящего пульта из _remote_costs[brand]:

Бренд	1 канал	До 5 каналов (с LED)	До 15 каналов
Simu	50 €	75 €	95 €
Somfy	65 € (Telis 1 RTS)	95 € (Situo 5 RTS)	125 € (Telis 16 RTS)
Decolife	35 €	55 €	70 €
Калькулятор берёт самый дешёвый пульт, чьё channels_max ≥ ch_need. Пример: для 3 маркиз с подсветкой нужно 4 канала → выбирается «до 5 каналов».

В смете отдельной строкой: «Пульт управления — <название>».

7. Ручное управление (без электропривода)
Просто строка «Ручное управление» по цене из _automation_eur.zip.manual_eur (по умолчанию 50 €).

8. Сборка
Сборка = (база + ткань + доплата за RAL + привод + пульт) × 8%

Процент берётся из _installation_costs.zip.assembly_pct = 0.08 (мы недавно снизили с 10% до 8%).

9. Установка (опционально)
Для ZIP формула монтажа особая — по площади проёма:

Установка = ширина × высота × 20 €/м² × euro_rate

То есть, например, ZIP 3 × 2.5 м = 7.5 м² × 20 € × 100 = 15 000 ₽.
Это _installation_costs.zip.price_per_sqm = 20.

10. Итоговая стоимость
Итого = база + ткань + цвет + привод + пульт + сборка + установка

Также рассчитываются варианты с НДС и безналом:

Наличные: = Итого
Безнал: Итого / 0.92
С НДС: Итого / 0.85
Что можно улучшить (если интересно)
Детализация моделей приводов — сейчас в строке просто «Электропривод Simu (ZIP)». Можно добавить конкретные модели по моменту/диаметру (как у локтевых).
Автоматическое предупреждение при ZIP 100 → ZIP 130 — сейчас переход тихий. Можно добавить пометку в смете.
Доп. датчики ветра/солнца — для ZIP сейчас выключены (проверка awning_type != "zip" в коде). Если нужны — добавлю.
