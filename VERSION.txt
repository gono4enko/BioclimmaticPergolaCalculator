Version: 4.5.8

Дата обновления: 12.04.2025
Основные изменения:
- Расширено управление галереей с новыми функциями для администрирования:
  • Добавлена надежная проверка целостности файлов изображений
  • Исправлена ошибка UnidentifiedImageError при отображении испорченных изображений
  • Реализована защита от сбоев при работе с поврежденными файлами
  • Добавлена возможность навсегда удалить изображения из системы
  • Создана специальная секция для обработки проблемных HEIC файлов
  • Реализована автоматическая конвертация HEIC в JPEG для корректного отображения
  • Оптимизирован интерфейс администрирования с группировкой кнопок действий
  • Добавлены кнопки "Опубликовать", "Исключить" и "Удалить" для управления
  • Реализован механизм создания резервных копий при удалении файлов
  • Добавлены информативные индикаторы статуса изображений (Активно/Исключено)

- Улучшения базового интерфейса:
  • Удалена кнопка "Перейти к результатам расчета" как избыточный элемент
  • Оптимизирован код, улучшена чистота интерфейса и удобство использования
  • Реализована мгновенная прокрутка к результатам после нажатия на кнопку расчета
  • Переименованы варианты подсветки на "LED-подсветка" и "RGB-подсветка"
