Version: 4.6.0

Дата обновления: 12.04.2025
Основные изменения:
- Улучшена система работы с изображениями:
  • Добавлена продвинутая проверка и автоматическое обнаружение дубликатов
  • Реализована надежная конвертация HEIC в JPEG с сохранением качества
  • Добавлен механизм сохранения исходных имён файлов при конвертации
  • Улучшена обработка ошибок при загрузке и обработке изображений
  • Добавлены информативные сообщения о процессе обработки изображений

- Расширены функции администрирования галереи:
  • Добавлен инструмент поиска дубликатов по имени файла и содержимому
  • Реализованы групповые действия для быстрой очистки дубликатов
  • Улучшен алгоритм распознавания дубликатов с проверкой содержимого
  • Добавлена возможность сохранять одно изображение из группы дубликатов
  • Добавлена функция быстрого выбора активных/исключенных изображений
  • Реализован удобный интерфейс групповых операций с выпадающим списком
  • Добавлена статистика состояния галереи (общее/активные/исключенные)

- Предыдущие улучшения:
  • Надежная проверка целостности файлов изображений
  • Исправлена ошибка UnidentifiedImageError при отображении испорченных изображений
  • Реализована защита от сбоев при работе с поврежденными файлами
  • Добавлена возможность навсегда удалить изображения из системы
  • Оптимизирован интерфейс администрирования с группировкой кнопок действий
  • Добавлены информативные индикаторы статуса изображений (Активно/Исключено)
