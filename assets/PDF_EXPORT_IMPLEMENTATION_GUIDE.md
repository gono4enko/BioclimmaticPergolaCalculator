# Руководство по реализации экспорта PDF

## Описание реализации
В новой версии PDF-экспорта мы добавили следующие улучшения:

1. **Профессиональная шапка с синим фоном и белым текстом**
   - Синий цвет фона (#2A4C7F) соответствует фирменному стилю компании
   - Белый текст для максимальной читаемости на синем фоне
   - Разделительная линия под шапкой для визуального отделения

2. **Логотип компании**
   - Включение логотипа в шапку PDF для профессионального вида
   - Логотип создан в формате SVG для качественного масштабирования
   - Автоматическая конвертация SVG в PNG с помощью CairoSVG

3. **Улучшенная контактная информация**
   - Правильное форматирование адреса и контактных данных
   - Структурированное размещение для легкого восприятия
   - Корректное отображение в правой части шапки

4. **Исправление проблемы с размерами перголы**
   - Решена проблема отображения дефолтных размеров (3.0×4.0м)
   - Теперь в PDF корректно отображаются выбранные пользователем размеры
   - Добавлено округление размеров для более аккуратного отображения

## Техническая реализация

### Логотип
- Создан в формате SVG для масштабирования
- Логотип размещен в директории `assets/logo.svg`
- Для PDF используется растровая версия (PNG)
- Конвертация SVG в PNG с использованием библиотеки CairoSVG

### Шапка PDF
- Метод `header()` в классе `PDF` (pdf_generator_fpdf_rus.py)
- Синий прямоугольник на всю ширину страницы
- Белый текст и разделительная линия
- Добавление логотипа или текстовой альтернативы, если логотип недоступен

### Обработка размеров
- Исправлена передача размеров из выбранных пользователем в pdf_data
- Добавлено округление размеров для более аккуратного отображения
- Логирование всех этапов для отслеживания корректной передачи значений

## Инструкция по обновлению
1. При необходимости обновить логотип, замените файл `assets/logo.svg`
2. Запустите скрипт `assets/convert_svg_to_png.py` для обновления PNG версии
3. Тест выполняется с помощью `assets/test_pdf_header.py`
4. Проверьте созданный PDF для подтверждения корректного отображения
