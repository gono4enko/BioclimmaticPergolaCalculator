Код для встраивания калькулятора на сайт

Инструкция по встраиванию

Чтобы встроить калькулятор перголы на ваш сайт или платформу Tilda, выполните следующие шаги:

  1. Скопируйте код HTML из блока ниже
  2. Вставьте код в нужное место на вашем сайте
  3. Адаптируйте размеры iframe (ширину и высоту) под ваш дизайн

Примечание

Калькулятор автоматически адаптируется к размеру iframe и отправляет информацию о своей высоте, чтобы родительская страница могла автоматически подстроить размер.

Для правильной работы в Tilda, добавьте дополнительный скрипт, указанный ниже, в раздел "Заголовок/Подвал" вашего сайта.

HTML-код для встраивания

<iframe 
    src="{{ url_for('main.iframe', _external=True) }}" 
    width="100%" 
    height="700" 
    id="pergola-calculator-iframe" 
    frameborder="0" 
    scrolling="no"
    style="overflow:hidden">
</iframe>

<script>
// Скрипт для автоматической настройки высоты iframe
window.addEventListener('message', function(event) {
    if (event.data && event.data.type === 'iframe-height') {
        const iframe = document.getElementById('pergola-calculator-iframe');
        if (iframe) {
            iframe.style.height = event.data.height + 'px';
        }
    }
});
</script>

Дополнительный скрипт для Tilda

function resizePergolaiFrame() {
    const iframe = document.getElementById('pergola-calculator-iframe');
    if (iframe) {
        iframe.style.height = (iframe.contentWindow.document.body.scrollHeight + 50) + 'px';
    }
}

window.addEventListener('message', function(event) {
    if (event.data && event.data.type === 'iframe-height') {
        const iframe = document.getElementById('pergola-calculator-iframe');
        if (iframe) {
            iframe.style.height = event.data.height + 'px';
        }
    }
});

// Вызываем функцию при загрузке страницы и при изменении размера окна
window.addEventListener('load', resizePergolaiFrame);
window.addEventListener('resize', resizePergolaiFrame);
